# Kester Family Cookbook

This is the permanent cookbook for the Kester family.

The goal is not to collect every possible recipe. The goal is to preserve the recipes the family actually cooks, improves, and returns to.

## Cookbook Principles

- Quality over quantity.
- Improve existing recipes before adding new ones.
- Preserve how the Kester family actually cooks.
- Keep recipes practical for real family life.
- Make recipes easy to reuse in weekly meal planning.
- Support AnyList import from the published website when practical.

## Recipe Status

Recipes should generally move through this path:

1. Rotation idea
2. Tested family meal
3. Revised recipe
4. Permanent cookbook candidate
5. Published recipe
