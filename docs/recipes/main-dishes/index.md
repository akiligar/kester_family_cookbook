# Main Dishes

Main dishes are the core dinner recipes in the Kester Family Cookbook.

Use this page to browse main dishes by primary protein. This keeps the cookbook useful as it grows and makes weekly planning easier.

Some main dishes are meal systems, where dinner intentionally becomes a second meal or work box.

## Discovery Notes

As new recipes are added, place each one under its primary protein group and make sure the recipe metadata includes:

```yaml
category: main-dish
protein: beef | chicken | pork | seafood | turkey | vegetarian | mixed
```

If a recipe does not clearly belong to one protein group, use `mixed`.
