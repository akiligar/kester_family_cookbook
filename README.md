# Kester Family Cookbook

Permanent GitHub-backed cookbook for the Kester family.

## Purpose

This repository stores polished, family-tested recipes in Markdown and publishes them through MkDocs. The cookbook should prioritize recipes the family actually cooks over generic internet recipe collection.

## Workflow

1. Develop and test meals in weekly rotation chats.
2. Promote only successful recipes into this permanent cookbook.
3. Store recipes as Markdown in `docs/recipes/`.
4. Publish with MkDocs through Cloudflare Pages.
5. Use the published recipe pages for AnyList imports when practical.

## Local Preview

```bash
pip install -r requirements.txt
mkdocs serve
```
